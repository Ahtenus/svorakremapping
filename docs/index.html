Svorak remapping

Svorak remapping är som ett lager mellan Windows och tangentbordet som översätter knapptryckningarna till Svorak. Perfekt att använda både hemma och borta eftersom ingen konfiguration i Windows ändras. Programmet är också konsekvent till skillnad från den inbyggda layoutväljaren.

Svorak Remapping
Version: 0.60.2
  • Portabelt
    Ingen installation, kan köras som icke administratör.
  • Svorak A1 och A5
    symboler under Alt Gr
  • Qwerty eller Svorak när Ctrl/Alt/Win är nedtryckt.
    Du kan alltså ha Ctrl + C på samma ställe som förut.
  • Kortkomando för att växla mellan Qwerty och Svorak
  • Gör onödiga tangenter användbara.
    T.ex. så kan du använda Caps Lock som en extra backspace.

Hur startar jag programmet automatiskt med Windows?

Windows 7 och Vista

Skapa en genväg till programmet i C:\Users\[Användarnamn]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup

Windows XP

Skapa en genväg till programmet och dra den till mappen Autostart under Alla program på startmenyn.

Hur får jag programmet att starta med Qwerty?

Pausa (Shift+F12) och välj sedan Starta i detta läge i trayikonens meny.
